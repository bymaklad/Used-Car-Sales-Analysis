## Project Overview
This project provides a full analytical study of a used car sales dataset using Power BI.  
The workflow includes data cleaning, normalization, modelling, DAX calculations, and dashboard development.
The final dashboard delivers insights into manufacturers, car models, engine sizes, fuel types, and year-based availability trends.  
The goal is to transform raw data into meaningful business insights that support decision-making.
## Skills Demonstrated
- Data Cleaning and Transformation using Power Query  
- Data Normalization and Star Schema Modelling  
- Creating Dimension and Fact Tables  
- Writing DAX Measures for KPIs  
- Designing Interactive Power BI Dashboards  
- Extracting Insights and Building Data Narratives  
- UI/UX Best Practices in Dashboard Design  
All these skills were learned and applied through the **Digital Egypt Pioneers Initiative (DEPI)**, which provided structured training and real-world analytical practice.
